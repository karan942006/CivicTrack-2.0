# MASTER APP GENERATION PROMPT

Build a production-ready Android application called "CitizenOS" using Kotlin, Android Studio, Firebase, Material Design 3, MVVM Architecture, Repository Pattern, Jetpack Navigation, Coroutines, StateFlow, ViewModel, Hilt Dependency Injection, Firebase Authentication, Firestore Database, Firebase Storage, Firebase Cloud Messaging, Google Maps SDK, Gemini AI Integration, Room Database for offline caching, and Clean Architecture principles.

The application should be designed as a next-generation AI-powered Smart Municipality and Digital Governance Super App that digitizes every interaction between citizens and local government.

The design must feel like a premium government platform similar to modern fintech applications, with beautiful animations, smooth transitions, dark mode, accessibility support, multilingual support, and a highly professional user experience.

---

MISSION

Create a platform that:

* Eliminates paper-based government processes.
* Increases transparency.
* Reduces corruption.
* Improves citizen participation.
* Accelerates public service delivery.
* Makes local government accessible from mobile devices.
* Provides real-time communication between citizens and authorities.
* Uses AI to simplify governance.
