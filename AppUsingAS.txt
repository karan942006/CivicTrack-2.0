That's a great choice. Using **Kotlin + Android Studio + Firebase**, you can build a professional Municipality App without needing a separate backend server initially.

# Recommended Tech Stack

### Frontend

* Kotlin
* Android Studio
* Material Design 3
* Jetpack Compose (recommended)

### Backend

* Firebase Authentication
* Cloud Firestore
* Firebase Storage
* Firebase Cloud Messaging (FCM)

### Maps

* Google Maps SDK

### Architecture

* MVVM
* Repository Pattern
* Hilt Dependency Injection
